Getting started

  1. Get started with Bulma
  2. Download the bulmaswatch.min.scss for the respective theme
  3. Replace bulma.css with bulmaswatch.min.scss

Use NPM

  1. Install package npm install bulmaswatch
  2. Import the respective theme from node_modules/bulmaswatch/<themename>
